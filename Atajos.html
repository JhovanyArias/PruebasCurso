Atajos para Visual Studio Code

  • lorem(x) = Crea parrafo con (x) de letras
  • etiqueta(x) + Tab = Crear x etiquetas
  • Alt + Shit + Arriba = Copia la linea y la pega arriba
  • Alt + Shit + Abajo = Copia la linea y la pega abajo
  • Ctrl + Alt = Mueve cursos al mismo espacio en las lineas de arriba o abajo
  • Alt + Tecla arriba o Tecla abajo = Mover linea arriba o abajo
  • Ctrl + } = Comentar/Descomentar Linea
